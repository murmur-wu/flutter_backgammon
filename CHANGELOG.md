## [1.0.0] - 2024-01-01

### Added
- Initial release of Gomoku mobile app
- 15x15 Gomoku board with Player vs Computer AI
- Heuristic AI with win/block/score evaluation
- Undo functionality (reverts player + AI move pair)
- Restart functionality
- Home screen with rules description
- Game screen with status bar, board, result banner, and controls
- Riverpod state management
- Unit tests for game rules, board, and AI
- Widget tests for screens
- GitHub Actions CI workflow
- iOS and Android platform support
